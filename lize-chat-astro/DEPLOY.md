# 部署说明

## 问题诊断

如果网站 https://www.lize.chat 上没有显示文章列表，可能是以下原因：

1. **部署的是旧版本** - 需要重新构建并部署最新版本
2. **浏览器缓存** - 清除浏览器缓存后刷新
3. **CSS 样式未加载** - 检查 CSS 文件是否正确加载

## 解决方案

### 1. 重新构建项目

```bash
cd lize-chat-astro
npm run build
```

构建完成后，检查 `dist/index.html` 文件，确认文章列表已包含在内。

### 2. 部署最新版本

将 `dist/` 目录下的所有文件部署到服务器。

### 3. 验证部署

访问 https://www.lize.chat/#dialogues，应该能看到：
- "最新讲会" 标题
- 文章卡片列表（5篇文章）

## 如果问题仍然存在

1. **检查构建输出**：
   ```bash
   npm run build
   ```
   查看是否有错误或警告

2. **检查文章文件**：
   确保 `src/content/blog/` 目录下的所有 `.md` 文件都有正确的 frontmatter

3. **检查内容集合配置**：
   确保 `src/content/config.ts` 中的 `blog` 集合配置正确

4. **查看浏览器控制台**：
   打开浏览器开发者工具（F12），查看 Console 和 Network 标签，检查是否有错误
